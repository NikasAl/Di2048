# Политика конфиденциальности — «2048 Цифры»

**Дата вступления в силу:** 20 июня 2026 г.

Настоящая Политика конфиденциальности описывает, как разработчик приложения «2048 Цифры» (далее — «Приложение», package name `ru.electronikas.diagonal`) обрабатывает данные пользователей. Приложение распространяется через магазины приложений RuStore и Google Play.

## 1. Какие данные собираются

### 1.1 Данные, собираемые Приложением напрямую

**Приложение НЕ собирает, не хранит и не передаёт разработчику следующие категории данных:**
- Персональные данные (ФИО, email, номер телефона, адрес)
- Финансовые данные (номера карт, платёжная информация)
- Контактные данные
- Сообщения пользователя
- Фотографии и изображения пользователя
- Геолокация

**Приложение хранит локально на устройстве пользователя (и не передаёт на серверы разработчика):**
- Текущее состояние игры (расположение плиток на доске) — для возможности продолжить игру после закрытия
- Лучший счёт для каждого размера игрового поля
- Настройки (уровень громкости звука, выбранный размер поля)

Эти данные хранятся в локальных настройках Android (`SharedPreferences`) и удаляются при удалении Приложения.

### 1.2 Данные, собираемые рекламным SDK

Приложение использует **Yandex Mobile Ads SDK** (Yandex Advertising Network) для показа рекламы (баннер, межстраничная реклама, реклама за вознаграждение). Yandex Mobile Ads SDK может собирать следующие данные:

- **Рекламный идентификатор (Advertising ID / GAID)** — используется для показа релевантной рекламы и оценки её эффективности
- **IP-адрес** — для передачи рекламных материалов
- **Информация об устройстве** — модель, операционная система, версия ОС, язык, размер экрана, оператор связи
- **Информация о сети** — тип подключения (Wi-Fi / мобильная сеть), оператор
- **Приблизительная геолокация** (на уровне страны/региона, по IP) — для таргетинга рекламы

Политика конфиденциальности Yandex Advertising Network: https://legal.yandex.com/partner_ch/

### 1.3 Данные, собираемые аналитическим SDK

Приложение использует **MyTracker SDK** (Mail.ru Group / VK) для агрегированной аналитики использования. MyTracker может собирать:

- **Рекламный идентификатор (Advertising ID / GAID)**
- **Идентификатор устройства** (Android ID)
- **IP-адрес**
- **Информация об устройстве** — модель, ОС, версия
- **События в приложении** — запуски, действия пользователя (например, нажатие кнопки «Отменить ход», «Убрать двойки»)
- **Приблизительная геолокация** (на уровне страны/региона, по IP)

Политика конфиденциальности MyTracker: https://tracker.my.com/legal/privacy/en

## 2. Как используются собранные данные

### 2.1 Цели обработки

- Показ и таргетинг рекламы через Yandex Mobile Ads
- Измерение эффективности рекламы (eCPM, показы, клики)
- Сбор агрегированной аналитики использования Приложения через MyTracker
- Улучшение пользовательского опыта на основе анонимной аналитики
- Защита от мошенничества и обеспечения безопасности

### 2.2 Передача третьим лицам

Разработчик **не передаёт** собранные данные третьим лицам, за исключением:
- Передачи данных в SDK Yandex Mobile Ads и MyTracker, как описано в разделах 1.2 и 1.3
- Требований применимого законодательства (по официальному запросу государственных органов)

### 2.3 Хранение данных

- Локальные данные (состояние игры, настройки) хранятся на устройстве пользователя до момента удаления Приложения
- Данные, собираемые SDK, хранятся в соответствии с политиками соответствующих сервисов (см. ссылки в разделах 1.2 и 1.3)

## 3. Реклама и персонализация

### 3.1 Рекламные форматы

Приложение показывает следующие форматы рекламы:
- **Баннер** — внизу экрана во время игры
- **Межстраничная реклама (Interstitial)** — после завершения партии (с ограничением частоты: не чаще одного показа на три завершённые игры, не чаще одного раза в 2 минуты)
- **Реклама за вознаграждение (Rewarded)** — по инициативе пользователя, при нажатии кнопок «Отменить ход» или «Убрать двойки»

### 3.2 Персонализация рекламы

Реклама в Приложении может быть персонализирована на основе данных, собираемых Yandex Mobile Ads, в соответствии с настройками персонализации рекламы пользователя на устройстве.

Пользователь может отключить персонализацию рекламы:
- **На устройстве Android:** Настройки → Google → Реклама → «Отключить персонализацию рекламы» (опция «Сбросить рекламный идентификатор»)
- **Через настройки Yandex:** https://yandex.com/adv/personalized_advertising

## 4. Права пользователя

### 4.1 Управление разрешениями

Пользователь имеет право:
- Отказаться от использования Приложения в любое время
- Управлять рекламным идентификатором через настройки устройства
- Отключить персонализацию рекламы (см. раздел 3.2)
- Удалить Приложение, что приведёт к удалению всех локальных данных

### 4.2 Доступ и удаление данных

- Локальные данные (состояние игры, настройки): пользователь может удалить их, удалив Приложение
- Данные, собранные SDK: пользователь может запросить доступ/удаление у соответствующих сервисов:
  - Yandex: https://yandex.com/support/privacy/requests.html
  - MyTracker: через форму обратной связи VK

### 4.3 Права согласно GDPR (для пользователей в ЕС) и ЗоПД (для пользователей в РФ)

Пользователи имеют право:
- На доступ к своим персональным данным
- На исправление неточных данных
- На удаление данных («право на забвение»)
- На ограничение обработки
- На портабельность данных
- На отзыв согласия

Для реализации этих прав в отношении данных, собираемых Yandex Mobile Ads и MyTracker, пользователь должен обратиться напрямую к соответствующим сервисам (см. раздел 4.2).

## 5. Безопасность данных

Разработчик применяет разумные технические и организационные меры для защиты локальных данных пользователя. Однако ни один метод передачи через интернет или электронного хранения не является 100% безопасным, поэтому мы не можем гарантировать абсолютную безопасность.

## 6. Дети

Приложение **не предназначено** для детей младше 13 лет и не адресовано им. Разработчик не собирает намеренно личные данные детей. Если вы считаете, что ребёнок предоставил нам личные данные, свяжитесь с нами (раздел 8), и мы удалим такую информацию.

## 7. Изменения в Политике конфиденциальности

Разработчик может обновить настоящую Политику конфиденциальности в любое время. Обновлённая версия будет опубликована на этой же странице с обновлённой датой вступления в силу. Рекомендуем периодически проверять эту страницу на наличие изменений.

## 9. Краткая сводка (TL;DR)

| Что | Где хранится | Кто имеет доступ |
|---|---|---|
| Состояние игры, рекорды, настройки | На устройстве (SharedPreferences) | Только пользователь |
| Рекламный идентификатор, IP, инфо об устройстве | Передаётся в Yandex Mobile Ads | Yandex |
| События в приложении, инфо об устройстве | Передаётся в MyTracker | VK / Mail.ru |

| Финансовые данные | **Не собираются** (биллинг удалён в P0) | — |
| Геолокация | **Не собирается** (только страна/регион по IP, для таргетинга рекламы) | Yandex, MyTracker |

Приложение не запрашивает и не использует разрешения `ACCESS_FINE_LOCATION`, `READ_CONTACTS`, `CAMERA`, `MICROPHONE`, `READ_EXTERNAL_STORAGE` или другие чувствительные разрешения.
